# Fundamentos del lenguaje

Los fundamentos del lenguaje constituyen la base sobre la cual se construye cualquier programa en Python. Antes de abordar conceptos más avanzados, es esencial comprender cómo se estructura un programa, qué reglas sigue la sintaxis del lenguaje, y cómo se declaran y utilizan elementos básicos como variables, operadores, comentarios y bloques de código.

## En este bloque aprenderás

- :material-laptop: **Introducción** — qué es Python, su historia y cómo funciona una computadora
- :material-tools: **Entornos de desarrollo** — instalar y configurar tu espacio de trabajo
- :material-database: **Tipos de datos** — números, cadenas de texto y datos primitivos
- :material-source-branch: **Control de flujo** — condicionales y bucles para tomar decisiones
- :material-table: **Estructuras de datos** — listas, tuplas, diccionarios, conjuntos y ficheros
- :material-puzzle: **Modularidad** — funciones, clases, excepciones y módulos
